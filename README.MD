🛠 Техническая концепция: "Ритм Биткоина"
Pulse — это облегченный форк Bitcoin Core, оптимизированный для работы в фоне на мобильных устройствах через Termux.

* 10-минутный Пульс: Узел «просыпается» только при нахождении нового блока в сети Bitcoin Mainnet. Это экономит 100% заряда батареи между блоками.
* Режим "Void": В периоды между пульсами судно находится в режиме скрытности (At Sea).
* Open Air Radio: Сообщения транслируются в открытый эфир. Ваша защита — не шифрование, а неизменяемость авторства, зафиксированная в глобальном хеш-дереве.
* Cargo Limit: Каждое сообщение ограничено 1 Кб. Это территория серьезных рапортов, а не цифрового спама.

------------------------------
📟 Командный интерфейс (TUI)
Взаимодействие с флотом происходит через консоль:

* /book @alias "btc_address" — Внести судно в записную книжку (Logbook).
* /echo — Пинг горизонта. Отвечают только те Капитаны, что есть в вашем списке контактов.
* /text @alias "Message" — Трансляция сообщения в эфир с привязкой к следующему пульсу.

Статусы Судна:

* [⚓ IN PORT] — Вы синхронизированы с последним блоком Биткоина. Логбук подтвержден.
* [🌊 AT SEA] — Вы в пути между блоками. Сообщения находятся в «трюме» ожидания.
